Google Chat / Webhook セットアップガイド
COMPLETE SETUP GUIDE

Google Chat
Webhook
追加方法

重要メール通知を Google Chat スペースに自動送信するための Webhook 設定手順を、画面のスクリーンショット付きでステップバイステップで解説します。

所要時間: 約5分難易度: 初心者向け

目次

  1. 01「通知用」スペースをクリック
  2. 02「アプリと統合」をクリック
  3. 03「Webhook を追加」をクリック
  4. 04 名前に「重要メール通知」と入力
  5. 05「保存」をクリック
  6. 06 Webhook のオプションメニューを開く
  7. 07「リンクをコピー」をクリック
1

「通知用」スペースをクリック

通知を受け取りたいスペースを開きます。

Google Chat のスペース一覧から「通知用」を選択

2

「アプリと統合」をクリック

スペース名の横にあるドロップダウンメニューから選択します。

メニューから「アプリと統合」を選択

3

「Webhook を追加」をクリック

「+ Webhook を追加」ボタンをクリック

4

名前に「重要メール通知」と入力

用途がわかる名前を付けましょう。

Webhook の名前欄に「重要メール通知」と入力

5

「保存」をクリック

「保存」ボタンをクリックして Webhook を作成

6

Webhook のオプションメニューを開く

保存した Webhook の右側にある「︙」をクリックします。

Webhook の右側にある三点メニューをクリック

7

「リンクをコピー」をクリック

コピーした URL を GAS やスクリプトに貼り付けて使います。

「リンクをコピー」で Webhook URL を取得

活用のコツ

📌
1スペース1用途 — 通知の種類ごとにスペースを分けると見やすい
🔒
Webhook URL は秘密情報 — 知っている人は誰でも投稿できるため漏洩注意
⚡
GAS と組み合わせ — Gmail の重要メールを自動で Chat に転送できる
